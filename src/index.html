Secure File Share

The Secure File Sharing System is a Flask-based platform designed to ensure safe and private file transfers. It provides end-to-end encryption to protect files during sharing and integrates user authentication to allow only authorized access. The system includes VirusTotal API integration, which scans uploaded files for potential threats before making them available to users. With its controlled access mechanism, users can securely upload, share, and retrieve files without the risk of unauthorized exposure. This platform emphasizes both data security and ease of use, ensuring a reliable solution for sensitive file management. It is built to maintain confidentiality, integrity, and trust in every file transaction.
